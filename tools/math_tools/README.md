# math_tools — 数学计算能力

算术运算工具。

## 工具

### add

计算两个数之和。

| 参数 | 类型 | 说明 |
|------|------|------|
| `a` | `float` | 加数 |
| `b` | `float` | 加数 |

**返回**：两数之和。

### calculate

对一个简单的算术表达式求值（安全沙箱，禁止代码注入）。

| 参数 | 类型 | 说明 |
|------|------|------|
| `expression` | `str` | 例如 `"1 + 2 * (3 - 4)"` |

**返回**：表达式的字符串化结果。

**允许的字符**：数字、空白、`+ - * / ( ) . %`

## 依赖

无（仅使用 Python 标准库）。
